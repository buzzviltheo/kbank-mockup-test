kbank 탭 라벨 치환 북마크릿

실제 백엔드는 그대로 동작. 탭 라벨만 아래 4가지로 실시간 치환.
참여 미션 → 1~100원
SNS 미션 → 100~1000원
콘텐츠 보기 → 천원~만원
보험 미션 → 만원이상

설치 & 사용법

  1. 북마크 바가 보이도록 켜주세요 (Chrome: Ctrl+Shift+B).
  2. 아래 검정 버튼을 북마크 바로 드래그해서 등록하세요.
    🧪 kbank 탭라벨 치환
  3. 스테이징 페이지를 브라우저에서 엽니다.
    https://pointhub-staging.buzzvil.com/kbank?p=...
  4. 위에서 등록한 북마크를 클릭하면 탭 라벨이 4개로 바뀝니다. React가 새 미션을 렌더링해도 자동으로 계속 치환됩니다.

드래그가 안 될 때 — 콘솔로 실행

스테이징 페이지에서 F12 → Console 탭에 아래 코드를 붙여넣고 엔터.
